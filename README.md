# CROCHETINGHUBB

## Project Overview
CROCHETINGHUBB is a web-based platform designed to support crochet sellers, hobbyists, and learners. The system provides a single digital space where users can explore handmade crochet products, learn crochet techniques through tutorials, and access useful crochet patterns. It helps small and home-based crochet businesses showcase and sell their products online easily.

## Features

### User
- Browse handmade crochet products
- Watch step-by-step crochet tutorials
- Access free crochet patterns
- Upload reference images to find similar patterns or tutorials
- Add products to cart and place orders
- Secure online payment

### Seller
- Seller registration and login
- Manage seller profile
- Upload crochet product images
- Add tutorials and patterns
- Manage and track customer orders

### Admin
- Manage users and sellers
- Manage products and tutorials
- Monitor orders and system activity
- Maintain overall system functionality

## Technologies Used
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP
